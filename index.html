Const vs Let

Const initializes constants, at this point you also need to assign a value to it.

Let, on the other hand, is a method of initializing a variable. Variables can have it's value changed/reassigned over the course of the code/runtime.

We are suposed to use Const more often, and use let only when specifying values that we know may get changed during runtime.
